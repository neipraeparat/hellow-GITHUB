GITсписок изучения

  • создать репозиторий в выбранной папке
    • настроить конфигурации репозитория
    • проверить соответствие ввода данных
  • начало работы с репозиторием

    • Разобраться с commit!!!

    • добавляем файлы в committed
    • вносим изменения в один или более файлов
    • коммитим его и добавляем с описанием в репозиторий
  • размещение репозитория на GITHUB

    • Разобраться с PUSH!!!

    • регистрируем аккаунт
    • создаем новый репозиторий в аккаунте
    • заходим в репозиторий и подключаем его удобным способом
    • закидываем файлы после коммита (git push -u)
  • Работа с ветвями проекта
    • Создание ветви "user" при помощи команды "checkout"
    • Проверка перехода на вторую ветвь разработки. Команда "branch"
    • Вносим изменения в проект в другой ветви
